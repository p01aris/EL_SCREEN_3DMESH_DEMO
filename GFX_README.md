# 3D Graphics Demo with Simple GFX Library

## 新增功能

### 简单图形库 (simple_gfx.h)
- **文字渲染**: 支持5x7点阵字体，包括大写字母、小写字母、数字
- **几何图形**: 圆形、矩形（填充或边框）
- **像素操作**: 设置/获取单个像素
- **可缩放文字**: 支持1x到任意倍数的文字缩放

### UI元素
1. **标题显示**: 屏幕顶部显示"3D DEMO"（3倍放大）
2. **装饰元素**: 四个角落的装饰圆圈
3. **信息显示**: 
   - 左下角：帧计数器
   - 右下角：网格大小信息
   - 底部中央：设备信息

### 技术特性
- **内存优化**: 字体数据存储在Flash中，避免RAM占用
- **高效渲染**: 内联函数确保最佳性能
- **单色显示**: 适合640x400单色屏幕

## 使用的字体
5x7点阵字体，包含：
- 数字 0-9
- 大写字母 A-Z  
- 小写字母 a-z
- 空格

## 渲染流程
1. 清空屏幕缓冲区
2. 绘制3D网格动画
3. 绘制UI元素和文字
4. 交换显示缓冲区

## 性能说明
- 字体渲染使用位操作优化
- 几何图形绘制使用经典算法（Bresenham直线、中点圆）
- 所有绘图函数都进行边界检查

这个简单的图形库为嵌入式系统提供了基本的2D图形功能，同时保持了良好的性能和较小的内存占用。
